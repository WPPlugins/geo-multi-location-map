Geo Map Locations plugin Display multi location with description and image on single page.

== Description ==
Geo Map Locations plugin having feature of Add,Edit, Delete locations, 
* Add location Title, Address, Description, latitude,longitude and image
* Get latitude,longitude value on Address
* You may select latitude,longitude based on your desired address like drag and drop location pointer on map
* Display google map preview instantly

Display your multiple locations on website's single page using simple shortcode. You can easily manage all your locations just like simple add/edit screen.

Geo location map setting
You may setting map size, radius, and your own API key which used to create and display map.


**Features** 

* Create customize marker title and image per Location.
* Create custom Map Zoom size and radius.
* Get coordinates on address.
* Get location preview in admin side.
* You can drag the marker in the editor to the get exact location on the map. Because sometime google make confuse.
* Widget zoom option.
* Settings for Map controls.
* Activate or Deactivate added location.
* Shortcode [geo_map] to insert map within your post or page.
* Select your preferred map type

== Installation ==

This section describes how to install the plugin and get it working.

1. Download and install the plugin from WordPress dashboard. You can also upload the entire “Geo Map Locations” folder to the `/wp-content/plugins/` directory

2. Activate the plugin through the ‘Plugins’ menu in WordPress

3. Go to Admin Menu "Geo Locations" menu display where you View, Add, Edit, Activate Deactivate locations and get shortcode to use on page and also used map setting.


== Frequently Asked Questions ==

= This plugin can insert multiple locations ? =
yes, Geo Map Location Plugin Insert multiple location on single map.


= Can I disable any location without delete ? =
yes, You can use activate/deactivate from plugin list page

= Can I use my own marker image for location pointer ? =
yes, When you insert or edit the location, you have option for "marker image", If you uploaded marker image than its display same else its display default.

= can I delete my uploaded marker Image ? =
yes, You can Delete your marker image from plugin list page .
